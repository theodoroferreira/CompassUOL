Questão 1:

O usuário deverá rodar o programa e escolher uma das opções disponíveis no console;
Se digitar 1, irá cadastrar 3 produtos na base de dados;
Se digitar 2, irá atualizar o produto 1 na base de dados;
Se digitar 3, irá remover o produto 2 da base de dados;
Se digitar 0, irá remover todos os produtos e finalizar o programa;
E, por fim, se digitar qualquer outro número, aparecerá uma mensagem dizendo que as únicas opções disponíveis vão do 0 até o 3.

Como solicitado, segue abaixo os prints da minha base de dados:

Print 1: Base de dados vazia (início do programa);
Print 2: Base de dados após inserção (opção 1);
Print 3: Base de dados após a alteração do produto 1 (opção 2);
Print 4: Base de dados após a remoção do produto 2 (opção 3);
